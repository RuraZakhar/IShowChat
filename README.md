IShowChat — Real-time Chat

Application URL: https://blue-bush-06f7b2303-preview.westeurope.7.azurestaticapps.net/

Features: 
          Real-time Messaging, AI Sentiment Analysis, Data Persistence, User Experience.

Tech Stack:
            Backend: ASP.NET Core 10.0
            Real-time: Azure SignalR Service
            Database: Azure SQL Database 
            Frontend: React (Vite) + Tailwind CSS 
            Deployment: Azure Web Apps & Azure Static Web Apps 

Security: All API keys, connection strings, and endpoints are managed via Azure Environment Variables and are not committed to GitHub.

Author: Zakhar Rura



Important Note
Since the backend and database are hosted on Azure's free tiers, the application may experience a cold start.
If the page takes a while to load or the chat doesn't connect immediately, please wait about 30-60 seconds for the server to "wake up" and then refresh the page.
